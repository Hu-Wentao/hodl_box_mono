# 韭菜罐子 (HODL Box) - Web 前端

韭菜罐子是一个基于 Next.js 构建的智能定投 Web 应用，集成了 AI 心理按摩功能，帮助用户坚持长期投资策略。

## 功能特性

- 💰 **资产概览** - 实时查看 USDT 和 BTC 余额
- ⏱️ **定投计划管理** - 创建、查看和管理您的定投计划
- 💬 **AI 心理按摩** - 当市场波动时，获得智能心理疏导
- 🔒 **安全存款/提款** - 简单便捷的资金管理

## 技术栈

- **框架**: Next.js 16
- **前端库**: React 19
- **样式**: Tailwind CSS
- **包管理**: pnpm

## 开始使用

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

应用将在 http://localhost:3000 启动

### 构建项目

```bash
pnpm build
```

构建产物将输出到 `dist` 目录

### 预览构建

```bash
pnpm start
```

## 项目结构

```
src/
  ├── app/           # 应用路由
  │   ├── page.tsx   # 首页
  │   ├── layout.tsx # 布局组件
  │   └── globals.css # 全局样式
  ├── components/    # 可复用组件
  └── utils/         # 工具函数
```

## 注意事项

- 目前使用模拟数据进行展示，实际部署时需要连接到真实的区块链接口
- AI 心理按摩功能需要连接到相应的后端服务
